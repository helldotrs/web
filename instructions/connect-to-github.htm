Connect Local Git to GitHub on Debian

1. Install Git:

sudo apt-get update
sudo apt-get install git

2. Set up Git:

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

3. Generate SSH key:

ssh-keygen -t rsa -b 4096 -C "your.email@example.com"

4. Add SSH key to SSH agent:

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa

5. Add SSH key to GitHub:

cat ~/.ssh/id_rsa.pub | xclip -selection clipboard

Copy the key to GitHub under Settings > SSH and GPG keys > New SSH key.

6. Test the SSH connection:

ssh -T git@github.com

7. Clone your GitHub repository:

git clone git@github.com:your-username/your-repository.git

8. Start working with your repository:

git add .
git commit -m "Your commit message"
git push origin master

Replace "Your commit message" with an appropriate commit message.
